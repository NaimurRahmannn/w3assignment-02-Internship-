Server
The site must be served through a HTTP server.
API Endpoints:
GET /get-property
This endpoint accepts the following query parameters:
most-popular
Dusting Off JS, TS & Node.js
13highest-price
lowest-price
limit
Functionality:
If most-popular=true , return the most_popular.json .
If highest-price=true , return the highest_price.json .
If lowest-price=true , return the lowest_price.json .
If limit is provided, return only the specified number of items
from the selected dataset.
Example Request:
/get-property?most-popular=true&limit=4
